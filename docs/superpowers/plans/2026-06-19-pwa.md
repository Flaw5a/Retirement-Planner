# PWA Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the retirement planner installable on an iPhone home screen and functional offline.

**Architecture:** Add `vite-plugin-pwa` to auto-generate a service worker and web manifest at build time; create a source SVG icon and convert it to required PNG sizes using `sharp`; add Apple-specific meta tags to `index.html`. No backend changes — purely additive build-time and static-asset changes.

**Tech Stack:** vite-plugin-pwa, sharp (icon generation), Vite 8, React 19

## Global Constraints

- Node 24, npm — do not use yarn or pnpm
- All icons live in `public/` and are committed to git
- Icon generation script runs once manually — it does NOT run at build time
- `registerType: 'autoUpdate'` — service worker must auto-update on new deploys
- App name in manifest: `Retirement Cash Flow Planner`, short name: `Retirement`
- Theme colour: `#111D35`, background colour: `#F4F6FA`

- [ ] **Step 3: Create the icon generation script**

Create `scripts/gen-icons.mjs` with this exact content:

```js
import sharp from 'sharp'
import { readFileSync } from 'fs'
import { mkdirSync } from 'fs'

mkdirSync('public', { recursive: true })

const svg = readFileSync('public/icon.svg')

const icons = [
  { file: 'public/apple-touch-icon.png', size: 180 },
  { file: 'public/pwa-192x192.png',      size: 192 },
  { file: 'public/pwa-512x512.png',      size: 512 },
]

for (const { file, size } of icons) {
  await sharp(svg).resize(size, size).png().toFile(file)
  console.log(`created ${file}`)
}
```

- [ ] **Step 4: Run the icon generation script**

Expected output:

### Task 2: Configure PWA plugin and Apple meta tags

**Files:**
- Modify: `vite.config.js`
- Modify: `index.html`

**Interfaces:**
- Consumes: `public/pwa-192x192.png`, `public/pwa-512x512.png`, `public/apple-touch-icon.png` from Task 1
- Produces: `dist/sw.js`, `dist/manifest.webmanifest` at build time — consumed by browsers/iOS

- [ ] **Step 1: Update vite.config.js**

Replace the entire content of `vite.config.js` with:

```js
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import { VitePWA } from 'vite-plugin-pwa'

export default defineConfig({
  plugins: [
    react(),
    VitePWA({
      registerType: 'autoUpdate',
      includeAssets: ['apple-touch-icon.png', 'pwa-192x192.png', 'pwa-512x512.png'],
      manifest: {
        name: 'Retirement Cash Flow Planner',
        short_name: 'Retirement',
        description: 'UK household retirement cash flow planner',
        theme_color: '#111D35',
        background_color: '#F4F6FA',
        display: 'standalone',
        scope: '/',
        start_url: '/',
        icons: [
          {
            src: 'pwa-192x192.png',
            sizes: '192x192',
            type: 'image/png',
          },
          {
            src: 'pwa-512x512.png',
            sizes: '512x512',
            type: 'image/png',
            purpose: 'any maskable',
          },
        ],
      },
      workbox: {
        globPatterns: ['**/*.{js,css,html,ico,png,svg,woff2}'],
        runtimeCaching: [],
      },
    }),
  ],
  test: {
    environment: 'node',
  },
})
```
